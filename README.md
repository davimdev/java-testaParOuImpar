# Projeto Básico Par ou Ímpar 🔢

Um programa simples em **Java** desenvolvido no Apache NetBeans que identifica se um número inteiro digitado pelo usuário é **par** ou **ímpar**.

## 🚀 Como funciona
O programa utiliza a classe `Scanner` para ler a entrada do usuário e aplica o operador de módulo (`%`) para verificar o resto da divisão por 2:
- Se o resto for 0, o número é **par**.
- Se o resto for 1, o número é **ímpar**.
- O código também possui uma validação básica para garantir que a entrada seja um número inteiro.

## 🛠️ Tecnologias
*   **Java** (JDK 17 ou superior recomendado)
*   **Apache NetBeans IDE**

## 📋 Como executar
1. Clone o repositório ou copie o código fonte.
2. Abra o projeto no **Apache NetBeans**.
3. Execute a classe `ProjetoParOuImpar.java`.
4. Digite um número no console quando solicitado.
